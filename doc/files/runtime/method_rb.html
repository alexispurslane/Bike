BikeMethod represents a method (also a function and lambda) in Bike. Example of usage:

BikeMethod.new(["a", "b"], ..., ..., ...)

This class is not meant to be directly used with its own instances of body, context, and vararg, becouse that is done in DefNode and LambdaNode and most of the instances for body, etc. are done by the parser You should rarely need to worry about this class
