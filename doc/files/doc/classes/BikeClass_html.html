        <div id='description'>

          <p>Classes are objects in Bike so they inherit from <a
          href="BikeObject.html">BikeObject</a>. <a
          href="BikeClass.html">BikeClass</a> is the way of representing Bike Classes
          that are initialized or not initialized. There are no properties in bike
          classes, only methods, which is conveiniently simple for syntax and code.</p>
        </div>
