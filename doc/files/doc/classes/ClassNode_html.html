        <div id='description'>

          <p>Defining a class is done in three steps:</p>
          <ol><li>
          <p>Reopen or define the class.</p>
          </li><li>
          <p>Create a special context of evaluation (set current_self and current_class
          to the new class).</p>
          </li><li>
          <p>Evaluate the body of the class inside that context.</p>
          </li></ol>

          <p>Check back how <a href="DefNode.html">DefNode</a> was implemented, adding
          methods to context.current_class. Here is where we set the value of
          current_class.</p>

          <p>Class definitions are stored into the following node. Once again, the
          `name` of the class and its `body`, a tree of nodes.</p>
