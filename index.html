Welcome to the Bike Homepage

As some people know, a bike is the most efficient man-powered vehicle known to man. The Bike Programming Language aims to be the same for programming.

Learn more »

Functional.

With lambdas, functions are values. With arrow function and everything being an expression, it becomes easy to use functional programming. With immutability by default, you can start functional programming today, painlessly. With a rich API of folds, maps, eaches, and many more of the hallowed functional programming capabilities, you'll never be at a loss, or have to resort to imperitive-land!

View details »

OOP, with a twist.

With algebraic datatypes, as well as classes and mixins, and everything being an object (even functions) this is a true OOP language. But, leaning twords the functinal side, we ditched inheritance intirely, and instead use mixins, hugely symplyfieing and empoewering the OOP side of Bike! And with Haskell-style algebraic datatypes to add to the mix, there'll be nothing you can't do!

View details »

Lightweight.

With some syntax borrowed from Ruby and Python, languages known for their lightweight syntax and ease of reading, Bike is a pleasent mix of curly brace style C syntax, and significant whitespace style languages, Bike is a pleasure to program in. And everything being an expresison, along with CoffeeScript style if expressions and until expressions, Bike is a buity to look at!
